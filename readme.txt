# Interactive Map Web App Documentation

## Introduction
Welcome to the documentation for the Interactive Map web app. This web app allows users to interact with a map, add markers, and customize their experience.

## Prerequisites
Before getting started, ensure you have the following prerequisites installed:
- Web browser with geolocation support
- Internet connection

## Installation
1. Clone the repository: `git clone https://github.com/PremsagarChakrapani/interactive-map.git`
2. Navigate to the project directory: `cd interactive-map`
3. Open `mp.html` in your preferred web browser.

## Usage
1. **Adding a Marker:**
   - Enter latitude and longitude in the form.
   - Provide a marker name in the "Marker Name" field.
   - Click the "Add Marker" button.

2. **Getting User Location:**
   - Click the "Get My Location" button to automatically fetch and display your current location on the map.

3. **Viewing Coordinates on Click:**
   - Click anywhere on the map to see the coordinates of the clicked location.

## Features
- **Interactive Map:** Explore the map by dragging and zooming.
- **Adding Markers:** Add custom markers with specific coordinates and names.
- **Geolocation:** Get your current location with the click of a button.
- **Click Coordinates:** View the coordinates of any location on the map by clicking.
